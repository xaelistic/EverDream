# EverDream VS Code Agent

This folder contains a base VS Code extension skeleton for the Everdream Dream Asset Pipeline.

## Included commands
- `everdream.enrich` — enrich a dream nugget into a poetic narrative.
- `everdream.generateAsset` — generate a dream asset folder with `narrative.md`, `asset.png`, and `metadata.json`.
- `everdream.remixStyle` — remix an existing dream asset into a new artistic style.
- `everdream.mintPrep` — export a mint preparation JSON file with provenance-ready metadata.

## Notes
- Free-tier generation is limited to one asset every 3 days.
- Premium upsell guidance is built into the asset generation workflow.
- Metadata includes Layer 1–4 provenance placeholders in line with Everdream requirements.

## Setup
1. Open the `vscode-agent` folder in VS Code.
2. Run `npm install`.
3. Run `npm run compile`.
4. Launch the extension by pressing `F5`.
